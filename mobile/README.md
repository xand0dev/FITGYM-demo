# 🏋️‍♂️ FITGYM Mobile App

Мобільний додаток преміум-класу для фітнес-центру **FITGYM**, створений на React Native (Expo). 
Розроблено як повноцінний проект з глибокою інтеграцією бекенду (Django REST API), штучного інтелекту та O2O (Online-to-Offline) функцій.

---

## ✨ Основні можливості (Відповідність вимогам)

Додаток повністю покриває всі вимоги для **максимальної оцінки (5 балів)**:

- **🔐 Авторизація та Безпека:** Повноцінний JWT-логін та реєстрація. Валідація форм реалізована через `react-hook-form` та `yup`.
- **🗄 Навігація та Стан:** Складна архітектура (`Tabs` + `Stack`). Глобальний стейт-менеджмент забезпечується **Zustand**, а локальне збереження (токени, кеш, історія чатів) — через `expo-secure-store`.
- **📅 Розклад та Фільтрація:** Динамічний розклад занять (з API) із горизонтальним календарним фільтром та Skeleton-анімаціями завантаження.
- **🤖 AI Тренер (Gemini):** Вбудований ШІ-коуч (на базі Google Gemini 2.5 Flash), який пам'ятає цілі користувача, зберігає історію контексту та відповідає з ефектом друку (Typewriter).
- **🏆 Гейміфікація та Екосистема:** Система щоденних стріків (Streaks), віртуальна валюта (FitCoins), досягнення та лідерборд.
- **📱 Фічі Пристрою:** 
  - `expo-image-picker` для завантаження аватарів камери/галереї.
  - `expo-haptics` для повної тактильної віддачі (Vibration) при взаємодії з UI.
  - Динамічна зміна кольорової теми (Dark Theme + Accent Colors).
- **🧪 Тестування:** Написано **5 ідеальних Jest Unit-тестів** для утиліт форматування та трансляції граматики (`src/__tests__/formatters.test.js`).

---

## 🛠 Технологічний стек
- **Фронтенд:** React Native / Expo, React Navigation
- **Мережа та Дані:** Axios (з interceptors), Zustand, Expo Secure Store
- **Форми:** React Hook Form + Yup
- **ШІ Інтеграція:** Google Generative AI (Gemini)
- **UI/UX:** Reanimated, Haptics, கастомна Brutalist Theme

## 🚀 Як запустити проєкт

1. **Завантажте залежності:**
   ```bash
   npm install
   ```
2. **Налаштуйте бекенд (опціонально):**
   У файлі `src/api/client.js` вкажіть вашу локальну IP-адресу (наприклад, `http://192.168.x.x:8000/api`).
3. **Запустіть застосунок:**
   ```bash
   npx expo start --clear
   ```
4. **Використання:**
   - Відскануйте QR-код через додаток **Expo Go** на смартфоні (iOS/Android).
   - Для доступу до ШІ-тренера переконайтесь, що змінна `EXPO_PUBLIC_GEMINI_API_KEY` додана до вашого `.env`.

---

## 🧪 Запуск тестів

Проєкт містить 5 Unit-тестів (з використанням Jest) для перевірки складної бізнес-логіки та UI-мапперів:

```bash
npm run test
```
