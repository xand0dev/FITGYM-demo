# 🏋️‍♂️ FITGYM Mobile App

Мобільний додаток для фітнес-центру **FITGYM**, створений на React Native (Expo). Додаток дозволяє переглядати розклад занять, бронювати тренування та відстежувати свій прогрес в особистому кабінеті.

## ✨ Основні можливості

- **Авторизація:** Мок-система входу з валідацією через `react-hook-form` та `yup`.
- **Розклад та фільтрація:** Перегляд списку доступних тренувань (FlatList) з можливістю пошуку та фільтрації за категоріями.
- **Деталі тренування:** Сторінка з описом конкретного заняття (інтенсивність, час, тренер).
- **Особистий кабінет:** Перегляд статистики (вага, кількість тренувань) та збережених записів. Глобальний стан працює через `zustand`.
- **Збереження сесії:** Використовується `expo-secure-store` для збереження стану авторизації після перезапуску.
- **Тестування:** Написано unit-тести для перевірки логіки глобального стору Zustand.

## 🛠 Технологічний стек
- **React Native / Expo**
- **Навігація:** React Navigation (Tabs + Stack)
- **Стан:** Zustand
- **Валідація:** React Hook Form + Yup
- **Локальне сховище:** Expo Secure Store
- **UI:** Brutalist theme (кастомні стилі), векторні іконки (`@expo/vector-icons`).

## 🚀 Як запустити проєкт

1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Запустіть проєкт:
   ```bash
   npx expo start
   ```
3. Відкрийте додаток:
   - Відскануйте QR-код через додаток **Expo Go** на вашому смартфоні.
   - Або натисніть `a` для запуску Android-емулятора (якщо він налаштований).
   - Або натисніть `w` для запуску веб-версії у браузері.

## 🧪 Запуск тестів
   ```bash
   npm run test
   ```
